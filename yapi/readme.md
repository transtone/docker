# docker-Yapi

Dockerized Yapi environment.

## usage

* 首次安装：  `docker-compose up install`
* 运行服务：  `docker-compose up web` or with `-d`
* 停止服务：  `docker-compose down`
* 更新 yapi：`docker-compose up update` 

## notice

* 使用端口 `3000:3000`
* yapi 版本 为 `v1.3.8`
